## Featured Projects

### [XAI_fer](https://github.com/rochatobias/XAI_fer)

Research project focused on Explainable AI applied to facial emotion recognition models. It explores attribution methods such as LIME, SHAP, and Grad-CAM to interpret machine learning and deep learning predictions.

**Technologies:** Python, Machine Learning, Deep Learning, XAI, LIME, SHAP, Grad-CAM

---

### [Planet Classifier](https://github.com/rochatobias/classifica-planetas)

Implementation of a multilayer perceptron model for planet classification based on astronomical data. The project focuses on practical machine learning concepts, data preparation, model training, and evaluation.

**Technologies:** Python, Scikit-Learn, Pandas, NumPy, Machine Learning

---

### [Background Modeling](https://github.com/rochatobias/background_modeling)

Implementation of a background modeling algorithm using frame averaging in MIPS-32 Assembly, with support for PGM and PPM image formats.

**Technologies:** MIPS-32 Assembly, Image Processing, PGM, PPM

---

### [Cache Memory Automation](https://github.com/rochatobias/CacheMemory-automation)

Performance analysis of cache memory behavior comparing row-major access, column-major access, and swapping, with automation through Shell Script.

**Technologies:** C, Shell Script, Linux, Computer Architecture

---

### [Blockchain Simulation](https://github.com/rochatobias/BlockChain-Simulation)

Simplified simulation of a blockchain inspired by Bitcoin, using persistent data structures and SHA-256 cryptography.

**Technologies:** C, Data Structures, SHA-256, Blockchain
